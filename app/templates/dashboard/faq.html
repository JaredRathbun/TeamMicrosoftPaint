Frequently Asked Questions

FAQ

Find the answers for the most frequently asked questions below

What is the purpose of the STEM Data Dashboard?

The STEM Data Dashboard was created to assist Merrimack's faculty with applying for grants and performing research. The dashboard was created to serve as a proof of concept of a new, innovative way to perform research.

How do I use the STEM Data Dashboard?

A user has many abilities on the dashboard. For example, to view all of the data in the database, visit the data page. To generate visuals based on the data, visit the visualizations page.

What devices does the Dashboard work on?

Currently, the dashboard works on all Laptops and Desktop computers. It has been tested on Chrome, Safari, and Edge. There is also some support for tablets such as iPads.

Can I upload my own data?

No, not as a viewer. If you would like to have access to add data, please reach out to an administrator, who can grant you data administrator privileges.

Can I access data for a specific student?

Yes, however the STEM Data Dashboard stores anonymized data. Because of this, Merrimack student ID numbers and names are not saved in the database.

Can I download a spreadsheet of the data?

Yes! Visit the data page and click on the "Download All Data" button at the top of the page.

Data Privacy Policy

The STEM Data Dashboard does not store personally identifiable information (PII). The data that is stored in the database is anonymized by the Merrimack College Provost's Office prior to being uploaded. Because of this, the dashboard does not store information such as the correct Merrimack student ID number, first names, or last names. Demographic data such as a student's home location, high school, race/ethnicity, gender, and SAT/ACT scores are stored in the database.

Dashboard Page

The dashboard page is designed to have important and commonly used data present and easy to access. For example at the top of the page, 4 common data values are shown. A screenshot of them is shown below:

Four boxes live below the top row of easily accessible data points. Each section has data visuals that you are allowed to view, interact with, and download. An example of the Average GPA Per Class Cohort graph can be seen on the left. For this specific visual, each bar in the chart is hoverable with the your mouse. To download this graph, follow these steps:

  1. Click on the three dots in the top right corner (as seen in the red box).
  2. A popup will appear, which looks like:

    1. To download the graph as a .pdf file, click on the "Download as PDF" button. This will immediately download the .pdf file to your browser.
    2. To download the graph's data as a .csv file, click on the "Download as CSV" button. This will immediately download the .csv file to your browser.

If you would like to download the graph as a .png image, click on the camera icon (as seen in the green box). A .png image will immediately be downloaded in your browser.

Please note that this camera icon is visible on any graph/chart that is generated with the STEM Data Dashboard. You will also see that there are many icons available next to the camera icon. If you hover over each icon, a tooltip will appear telling you what the respective button does.

Visualizations Page

The Visualizations page serves as the page on the dashboard where the most operations can be performed. On this page, you are able to specify the columns in the dataset you would like to analyze; along with the type of chart or graph you would like to generate. At the top of the page, you are presented with a general data bar which holds common analytics from the database. An example of the general data bar can be seen below.

Below the general data bar you will see 4 boxes, each with a label showing their respective operations. Each box is broken down into sections below, explaining how to use each section.

Class by Class Comparisons

This visualizations option requires you to input at least 4 pieces of data; 2 courses you would like to analyze, the column you would like, and the type of graph/chart. To generate a chart/graph follow these steps:

  1. Select 2 courses and semesters to analyze (as seen in the purple box). When the dropdown for Select course... is clicked, you will see a list of all the course codes in the database. Once a course is selected, the Select semester... dropdown to the right of the given course dropdown will populate with the available semester. Select a semester from that list.
  2. Click on the Select Data Column dropdown (as seen in the orange box). You will see a dropdown with a list of the column in the database.
  3. Click on the Select a Chart/Graph Type dropdown (as seen in the blue box). You will see at least 1 option for the type of graph/chart you can generate for the data you selected.
  4. Click on the Generate Chart/Graph button (as seen in the green box).
  5. You will be presented with a graph of the data you generated. An example of a generated bar chart can be seen below in the bottom image.

If you would like to add more than 2 courses to analyze, click on the Add New Course button (as seen in the red box above).

You will see a new option for a course/semester under the original 2 course/semester dropdowns. You can repeat this action to add as many classes as you would like, but you cannot have less than 2. Each set of dropdowns for a new course/semester will look similar to this:

COVID Data Comparisons

To the right of the Class by Comparisons box lives the COVID Data Comparisons box. Inside of this box, you are able to select a column to analyze, and only the data present during the COVID-19 Pandemic years will be shown.

To generate a COVID graph, do the following:

  1. Click on the Choose a column... dropdown (as seen in the red box).
  2. You will see a dropdown with all of the columns in the table on the data page. Select one of the columns (an example of this dropdown can be seen to the right).
  3. The Generate Chart/Graph button (as seen in the green box) will change from being disabled to enabled. When you click on this button, a chart/graph will pop up.
Bar Chart Generation

Below the Class by Class and COVID Data Comparison sections lives 2 more boxes. On the left side, the Bar Chart Generation section is visible. An example of what this box looks like can be seen below:

To generate a bar chart, follow these steps:

  1. Select an X-Axis column to compare. To do this, click on the X-Axis Choose a column dropdown (as seen in the red box). This will allow you to choose from any of the columns in the database.
  2. Select an Y-Axis column to compare. To do this, click on the Y-Axis Choose a column dropdown (as seen in the purple box). This will allow you to choose from any of the columns in the database.
  3. Click the Generate Bar Chart button (as seen in the green box). You will see a pop up with the generated bar chart appear.
